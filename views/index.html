URL Shortener

Example Usage:

If I wanted to change the link of https://www.freecodecamp.org
I'd write the link in the bar and the click on the Short URL button.
This will generate a number.

[this_project_url]/api/shorturl/[generated_number]

https://cooked-kindhearted-seeder.glitch.me/api/shorturl/3
